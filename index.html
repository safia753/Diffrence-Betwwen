Modern Web Development Concepts

1. What is React?

React is a JavaScript library developed by Facebook for building user interfaces. It is component-based, declarative, and uses a Virtual DOM to efficiently update the browser DOM.

// React component example (JSX)
function Hello() {
  return <h1>Hello, World!</h1>;
}

2. SPA vs MPA

SPA (Single Page Application) loads a single HTML page and updates content dynamically without refreshing the page. MPA (Multi Page Application) reloads the entire page on each navigation.

Feature SPA MPA
Speed Fast (after load) Slower
SEO Needs SSR/Workaround Better by default
Complexity Less More

4. Library vs Framework

A library offers specific functionality and you call it in your code. A framework provides the entire structure and calls your code instead (Inversion of Control).

5. DOM vs Virtual DOM

DOM (Document Object Model) represents the structure of a webpage. Modifying it directly can be slow.

Virtual DOM is a lightweight copy that updates the real DOM only when needed, improving performance (used by React).
